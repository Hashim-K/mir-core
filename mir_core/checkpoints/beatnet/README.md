# BeatNet Baseline Checkpoints

`model_1_weights.pt` is the official BeatNet model-1 baseline checkpoint from:

https://github.com/mjhydri/BeatNet/blob/main/src/BeatNet/models/model_1_weights.pt

The upstream BeatNet README identifies model 1 as the GTZAN-trained model.
